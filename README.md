# 🧩 Puzzle Game

A fun and interactive Puzzle Game built using **HTML, CSS, and JavaScript**. Players can solve image-based puzzles across multiple levels while tracking their progress through achievements and a leaderboard.

## 🚀 Features

- 🎮 Multiple puzzle levels
- 🖼️ Image-based sliding puzzle
- 🏆 Leaderboard
- 🎖️ Achievement system
- 👤 User Profile
- 📊 Dashboard
- 📱 Responsive design
- 🔄 Restart and Shuffle options
- 💾 Progress saving

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Swerudra/Puzzle-Game.git
```

2. Open the project folder.

3. Run `index.html` in your web browser.
